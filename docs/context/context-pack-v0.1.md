# Context Pack v0.1

## Produto

O Assistente de Avaliação de Mérito Tecnológico é uma aplicação privada para apoiar especialistas na análise de projetos de inovação.

## Papel da IA

A IA poderá futuramente extrair, organizar, recuperar e sintetizar evidências dentro de limites explícitos. Não poderá aprovar projetos, inventar informações ou substituir a decisão humana.

## Premissas

- dados podem ser confidenciais;
- regras variam por programa;
- cada conclusão deve ser rastreável;
- ausência de informação deve permanecer visível;
- o sistema deve funcionar sem provedor externo.

## Perfil inicial

O primeiro perfil é `finep_mais_inovacao_tecnologias_digitais`, desacoplado do nome do produto e do pacote central.

## Precedência

Regulamento e erratas → anexos → FAQ → treinamento oficial → documentos da proposta → fontes externas autorizadas → inferências → redação gerada.
